	СЕКРЕТЫ КРАСОТЫ ОДНОЙ СТРОКОЙ<br><br>1. Для борьбы с морщинами смазывайте лицо медом.<br><br>2. Полезно перед сном наносить вокруг глаз касторовое масло.<br><br>3. При выпадении волос втирайте в кожу головы настой лопуха.<br><br>4. Остатками выжатого лимона смазывайте руки и локти.<br><br>5. При шершавости кожи рук втирать в шершавые места крахмал.<br><br>6. В качестве пудры можно использовать небольшое количество крахмала.<br><br>7. Для подсушивания жирных волос каждый день или через день втирайте в кожу головы заварку черного чая.<br><br>8. Полезно делать пилинг тела из кофейной гущи, необходимое количество которой можно собрать за несколько дней.<br><br>9. Свежевыжатый сок облепихи рекомендуется в качестве маски для сухой кожи (1-1,5 столовой ложки).<br><br>10. Волосы приобретут цвет красного дерева, если при окрашивании в упаковку хны добавить 3-4 ложки какао.<br><br>11. При облысении каждый день втирать в кожу головы масло жожоба.<br><br>12. Настой цветков бузины отбеливает, смягчает и тонизирует сухую кожу.<br><br>13. Для увлажнения сухой и чувствительной кожи на ночь нанести на лицо масло жожоба, смешанное с небольшим количеством воды.<br><br>14. При потемнении ногтей регулярно за полчаса до приема пищи пить по одной столовой ложке огуречного рассола.<br><br>15. Для улучшения цвета лица делать маски из смеси меда и соли, которые оказывают омолаживающее действие.<br><br>16. Эфирное масла лаванды и чайного дерева являются естественными антисептиками, которые помогают бороться с прыщами.<br><br>17. Утром и вечером протирать лицо чаем, заваренным в минеральной воде с кусочком сахара. Подходит для любой кожи.<br><br>18. Регулярное питье мятного чая придает коже лица свежесть и розоватый оттенок.<br><br>19. Полезно смазывать брови и ресницы касторовым маслом или смесью касторового масла с ромом.<br><br>20. Для глубокой чистки кожи лица использовать взбитый яичный желток и лимонный сок. Наносить на лицо слоями.<br><br>21. Для смягчения рук втирайте в них смесь из равных частей уксуса и растительного масла.<br><br>22. Перед сном смазывать лицо и руки смесью из равных частей оливкового масла и лимонного сока.<br><br>23. Для очищения кожи лица применяйте смесь сухой овсянки с водой и простым йогуртом.<br><br>24. При прыщах хорошо помогает маска из тертого яблока и небольшого количества меда.<br><br>25. При беременности смазывать живот маслом какао во избежание появления растяжек.<br><br>26. Свежий сок каланхоэ разбавить водой (1:3) и делать примочки при аллергии.<br><br>27. Народное иранское средство для укрепления волос - маска из бесцветной хны, разведенной теплым кефиром.<br><br>28. Для укрепления волос втирают разварившуюся гречку.<br><br>29. Для тонких волос делают маску из взбитого одного желтка и столовой ложки растительного масла.<br><br>30. При выпадении волос за час до мытья втирайте в кожу головы смесь из равного количества касторового масла и рома.<br><br>31. Для укрепления волос после каждого мытья в кожу головы втирают настой крапивы (1 ст. ложка сухой крапивы на 200 г кипятка.)<br><br>32. Чтобы волосы были сильными и шелковистыми, их моют отваром из корней аира, лопуха и листьев крапивы (всё в равных количествах).<br><br>33. Массаж лица серебряными ложками: ложки погружают в травяные отвары разной температуры и похлопывают по лицу по специальным линиям. Эффект невероятный!Секреты красоты одной строкой
