	1) Создадим текстовый документ (.txt) и назовем его warmup. <br>2) Скопируйте и вставьте в него следующее: <br><br>sv_cheats 1 <br>bot_kick // Кикает ботов <br>mp_autoteambalance 0 // Отключается баланс команд <br>mp_limitteams 0 // Отключается лимит в командах <br>mp_startmoney 16000 // Выдается 16000$ вначале раунда <br>mp_round_restart_delay 0 // Отключается задержка вначале раунда <br>mp_freezetime 0 <br>mp_buytime 3600 // Увеличивается время закупки <br>mp_do_warmup_offine 1 // <br>mp_do_warmup_period 1 // Данные 3 команды обеспечивают долгую разминку <br>mp_warmuptime 36000 // <br>bind z "give weapon_flashbang; use weapon_flashbang" // Бинд на выдачу флешки <br>bind x "give weapon_smokegrenade; use weapon_smokegrenade" // Бинд на выдачу дым <br>bind c "give weapon_molotov; use weapon_molotov" // Бинд на выдачу молотова <br>bind v "give weapon_incgrenade; use weapon_incgrenade" // Бинд на выдачу зажигательной гранаты <br>bind f1 "noclip" // Включение/отключение полёта <br>sv_grenade_trajectory 1 // Показывает траекторию полёта гранаты <br>ammo_grenade_limit_default 30 // Лимит гранат <br>ammo_grenade_limit_flashbang 30 // Лимит флешек <br>ammo_grenade_limit_total 30 // Общий лимит гранат <br>sv_showimpacts 1 // Цветные отверстия от пуль <br>sv_infinite_ammo 1 // Бесконечные патроны<br><br>3) Нажимаем "Сохранить как", пишем названием warmup.cfg (именно с .cfg) и сохраняем. <br><br>После того как Вы сохранили конфиг, его нужно поместить в папку с игрой. Для этого перейдите в директорию STEAM и далее в:<br>steamapps\common\Counter-Strike Global Offensive\csgo\cfg<br><br>4) Заходим в CS: GO <br>5) Создаем одиночную игру с ботами <br>6) Открываем консоль (ё - по стандарту) <br>7) Прописываем в консоли: exec warmup.cfg<br>Поздравляем, вы включили режим тренировки.<br><br>Так же Вы можете сделать бинд на активацию этого конфига. <br>Бинд будет выглядеть так: bind "L" "exec warmup.cfg" <br>Теперь созданный конфиг будет активировать при нажатии на буку "L".
